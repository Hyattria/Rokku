## 特性

- 单元测试覆盖率 90%+，提供稳定性保障
- 完善的中英文文档和示例
- 支持按需引入
- 支持主题定制
- 支持 TypeScript

## 📦 Init App

npm

```bash
npm install
```

yarn

```bash
yarn
```

## 🔨 使用

### 启动文档

```bash
yarn start
```

### 打包文档

```bash
yarn docs:build
```

### 发布文档

```bash
yarn docs:deploy
```

### 打包组件

```bash
yarn build
```

### 测试组件

```bash
yarn test
```
